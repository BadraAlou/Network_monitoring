Nos dépendances principales

Django

Le framework web Python sur lequel repose toute l’application.

Documentation pip install Django

Django REST Framework

Pour créer nos APIs d’alertes et d’événements.

Documentation pip install djangorestframework

python-nmap

Interface Python pour piloter Nmap directement.

PyPI pip install python-nmap

netifaces

Pour détecter automatiquement le sous-réseau local.

PyPI pip install netifaces

stripe

SDK pour intégrer le paiement par carte bancaire (test).

PyPI pip install stripe

django-tailwind

Plugin pour intégrer Tailwind CSS à Django.

GitHub pip install django-tailwind
